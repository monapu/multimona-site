I sent monacoin to MultiMona but they never arrived

What is the symptom ?

You sent some monacoin to MultiMona but the monacoin never arrived. There is no record of the transaction in MultiMona.

What can you do about it ?

The first thing to check is whether the monacoin was actually sent ok. You can do this as follows:

  1. Request the transaction id from the person or company that sent it to you.
  2. Check the transaction on Blockchain.info or Monacoin Block Explorer to be sure that it is actually on the Monacoin network.

I've verified that the transaction is on the blockchain and it's confirmed

To get the missing transaction into your wallet:

  1. Click on the wallet in the ' Wallets' side panel to make it the active wallet.
  2. Go into the 'Tools |  Reset Blockchain and Transactions' menu option.
  3. Click on the ' Reset blockchain and transactions' button and let MultiMona sync the blockchain. Just leave MultiMona to work its way through the sync.

You should then see the transaction in your wallet.

I'm running an old version of MultiMona (before 0.5.16)

Earlier versions of MultiMona did not show transactions that had inputs that were not confirmed. As this caused confusion this has been changed in version 0.5.16 so that these transactions are shown (unconfirmed). If you have an old version of MultiMona download the latest copy and install it. You will need to do a ' Reset blockchain and transactions' so that MultiMona can resync to see any transactions like this.
