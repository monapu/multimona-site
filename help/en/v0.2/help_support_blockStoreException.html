BlockStoreException

You see a BlockStoreException very occasionally when MultiMona starts. It prevents you synchronising with the Monacoin network.

You can fix this problem by doing the following:

  1. Find where MultiMona is storing your user data - see the help topic Troubleshooting

  2. Shut down MultiMona

  3. Go to your MultiMona user data directory - worked out in step 1 - and delete the file multimona.spvchain

  4. Start up MultiMona again

It will then regenerate this file and start syncing. After it has synced you should see all your monacoin in your wallet..
