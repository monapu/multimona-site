Upgrading MultiMona

New versions of MultiMona are released containing bug fixes and new features every so often. We recommend you keep your version of MultiMona up-to-date.

This guide tells you how upgrade to a newer version. It is assumed that you have reviewed the Getting Started guide and know where all your wallet files are.

Step 1 - Locate your wallet files

If you have a standard installation (i.e. you followed our installation notes) upgrading MultiMona should be a seamless process. Your wallet and configuration files will not be in the same directory as the MultiMona application.

Note: You can create new wallets in any directory however we recommend you store your wallets and configuration files separate from the installation directory.

You can always see the file location of your current wallet in the main title bar of your MultiMona window. For instance, it might say: "MultiMona - My wallet - /Users/jim/Library/Application Support/MultiMona/multimona.wallet". The file name at the end is where your wallet is stored.

Step 2 (optional) - Move your wallets

If you have stored your wallets in the MultiMona installation directory (this is not recommended) and want to move the wallets out of this directory then please review the Moving A Wallet article.

Step 3 - Run the installer/ Mac DMG file

Download the installer from the official MultiMona site.

Follow the standard installation instructions for your computer.

Step 4 - (Windows and Linux only) Run the old version's uninstaller

Once you are happy with the new version of MultiMona there is a uninstaller shortcut provided so that you can uninstall the older version of MultiMona.

This is not required on Mac OS X since MultiMona.app is simply overwritten.

Finish

That's all there is to it. MultiMona will alert you if there is a problem and the main help contents provide answers to many common questions.
