How do I password protect my wallet ?

There are two types of wallets in MultiMona. They are:

  • Non password protected wallets. They are shown with the icon ''.
  • Password protected wallets. Their icon is ''.

Password protected wallets are much more secure than non password protected ones. It is recommended that you password protect all MultiMona wallets.

You can add a password to a wallet using the 'File | Add Password' menu option.

MultiMona recommends using a password manager like LastPass or KeyPass to generate a one-off password with at least 14 random characters.

Do not lose the password that you use for a wallet!

You need your password to spend your monacoin. There is no way to recover your password if you lose it.
